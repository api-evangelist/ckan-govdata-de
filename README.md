# GovData Germany (ckan-govdata-de)

GovData Germany is a national government open-data portal for Germany running **CKAN**, exposing a standardized catalog API over ~151,707 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/ckan-govdata-de/refs/heads/main/apis.yml)

## Type
- **kind:** government  ·  **software:** CKAN  ·  **version:** 2.10.10

## API
- **GovData Germany CKAN Action API** — base `https://ckan.govdata.de/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://ckan.govdata.de)
- Live endpoint verified: `https://ckan.govdata.de/api/3/action/status_show`
